Overview

Mini Store Management System is a full-stack web application that allows users to manage products within a store inventory. 
The application provides complete CRUD functionality, enabling users to create, view, update, and delete products through an easy-to-use interface.

The project was built using HTML, CSS, Bootstrap, Java Spring Boot, and MySQL.

CRUD Operations
Create: Add new products to the inventory by entering product details.
Read: View all available products stored in the database.
Update: Edit product information such as name, description, price, stock quantity, and image URL.
Delete: Remove products from the inventory.
